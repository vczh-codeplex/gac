Introduction

GacUI library is released by source code in http://gac.codeplex.com/. GacUI is released under the MS-PL license. That says, if you want to redistribute your open source project that depends on GacUI, all GacUI source code (modified or not) should be under MS-PL, and your can choose your favorite license for your code. In other cases, MS-PL is very similar to Apache or BSD. Currently a complete GacUI release contains only 5 source files:

  • GacUIIncludes.h (includes memory.h and string.h)

    This is the recommended header file to include. This file contains all necessary content to use GacUI. All controls and necessary basic constructions are all in this file.

  • GacUI.h (includes Vlpp.h, Windows.h and Direct2D components)

    This is the full featured GacUI header files. The differece between GacUI.h and GacUIIncludes.h is that, GacUI.h contains some extra interfaces that required Windows.h. If GacUI.h is included, Windows.h is included as well. Besides Windows.h, GacUI.h also provides interfaces for accessing internal Direct2D or GDI objects for any GacUI window.

  • GacUI.cpp

    Implements of GacUI library.

  • Vlpp.h and Vlpp.cpp

    This file comes from http://vlpp.codeplex.com/ project, which constructions that encapsulated strings, containers, regex, threading, synchronization and other tools for making C++ development easier.

Download

The source code and the release of the GacUI project hosts in http://gac.codeplex.com/. We suggested the following two ways to get the source code:

  • Release

    Release contains well tested and stablized code, necessary tools and HTML documents. Currently the first release is making. It will be available soon.

    GacUI prepared online HTML documents, click HERE to begin.

  • The latest source code

    The latest source code contains fixings to bugs that found in the previous release, and new features, that under developing, for the next release. The latest source code doesn't promised to be well tested and stabilized.

    If you want to download the latest source code, please go to http://gac.codeplex.com/SourceControl/BrowseLatest, and click Download.

    When a zip file is successfully downloaded, extra the zip file, and those 5 source files will be found under Libraries\GacUI\Public\Source\ directory. Only release provides HTML documents. If you want to build the document yourself, follow the instructions:

    • Open Libraries\GacUI\GacUISrc\GacUISrc.sln using Visual Studio 2010.
    • Compile the GacUISrc project under Debug configuration.
    • Run Libraries\GacUI\Public\Release.bat in Windows Explorer.

    After these steps, the latest HTML documents will be generated in Libraries\GacUI\Public\Web\StaticHtmlDoc directory.
